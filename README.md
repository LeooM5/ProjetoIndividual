# ⚽ Minha Jornada como Goleiro no Futsal

Este projeto foi desenvolvido com o objetivo de demonstrar minha evolução ao longo do curso de Análise e Desenvolvimento de Sistemas, aplicando na prática conhecimentos de Front-end, Back-end e Banco de Dados.

Além do aspecto técnico, escolhi um tema que fez parte da minha vida: minha trajetória no futsal, com ênfase no clube Thomaz Mazzoni, onde vivi momentos importantes da minha infância, evolução esportiva e conquistas pessoais.

## 📌 Sobre o Projeto

O site apresenta minha história no esporte de forma organizada e interativa, mostrando minha evolução como goleiro e a importância do futsal no meu desenvolvimento pessoal.

Além da parte informativa, o sistema possui funcionalidades que permitem ao usuário:

- Criar uma conta
- Fazer login
- Registrar partidas
- Visualizar estatísticas e métricas em dashboard
- Acompanhar informações sobre desempenho no futsal

## 🛠️ Tecnologias Utilizadas

- HTML
- CSS
- JavaScript
- Node.js
- MySQL

## 🎯 Objetivo

O principal objetivo do projeto é unir aprendizado com algo que marcou minha vida, utilizando a programação para contar minha trajetória e demonstrar os conhecimentos adquiridos durante o curso.
